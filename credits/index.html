This site is built with the static site generator Hugo, and uses a customised theme based on Anatole by Alexander Bilz, itself based on farbox-theme-Anatole by Cai Cai. The paginator is based on an article by Glenn McComb and the Tweet formatter is by Bryce Wray.

The fonts are Roboto by Christian Robertson and Open Sans by Steve Matteson, both from Google Fonts.

The site source lives on Github. When updates are made an action automatically builds the site and pushes it to AWS S3.
